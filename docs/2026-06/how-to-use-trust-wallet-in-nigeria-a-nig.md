## 1. Installation & Account Setup

- Download Trust Wallet from official app store (iOS/Android)
- Create new wallet or import existing seed phrase
- Write down and securely store 12-word recovery phrase (never screenshot, never share)
- Set strong PIN/biometric lock

## 3. Asset Management

- Add custom tokens by contract address (verify on official docs)
- Monitor gas fees (higher during network congestion)
- Use BEP20/ERC20 networks based on your target blockchain
- Enable notifications for large transfers

## 4. Security Checklist

- [ ] Recovery phrase stored offline (hardware wallet, safe deposit box)
- [ ] PIN set (never 1234, birthdate, or sequential)
- [ ] Biometric lock enabled
- [ ] Public WiFi avoided for transactions
- [ ] Phishing links never clicked (verify URLs carefully)
- [ ] Recovery phrase never entered into online forms
- [ ] Backup PIN saved separately from phrase
- [ ] Regular balance reviews for unauthorized access

## 5. Common Issues & Solutions

**Slow Transaction**
- Check network congestion on blockchain explorer
- Increase gas fee if needed
- Wait 15+ minutes before investigating

**Failed Transaction**
- Verify wallet address format (checksum sensitivity on some networks)
- Check sufficient balance including gas
- Try on different network if available

**Tokens Not Showing**
- Add custom token via contract address
- Verify token is live on blockchain explorer
- Try toggling "hidden tokens" in settings

**Unable to Withdraw to Naira**
- Use peer-to-peer exchanges (Binance P2P, Luno)
- Verify identity requirements met
- Check withdrawal limits per day/month

## 6. Regional Considerations (Nigeria)

- CBN regulatory environment: crypto holdings permitted for individuals, check latest policy
- Most banks support transfers to exchanges; some flag crypto transfers
- P2P transfers often faster and safer than bank routing
- Tax reporting: keep transaction records (may be required for large amounts)
- Phone number verification critical (SIM swap attacks common in region)
- Use trusted peer-to-peer platforms only
- Consider VPN if ISP blocks exchanges
